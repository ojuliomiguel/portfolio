Que sou eu

Graduando em Ciência da Computação e atualmente trabalhando como desenvolvedor backend. Comecei na área de desenvolvimento de software dando suporte a um sistema legado em Delphi e com isso surgiu oportunidades para realizar integrações nesse sistema. Diante dos desafios - oportunidades - pude criar integrações com nodejs com sucesso. Assim, o desenvolvimento de soluções customizadas de integrações foram criadas expandindo-se até para criação de chatbots para whatsapp e telegram. Observo as tecnologias como ferramentas de apoio - um meio - para atingir objetivos. Para mim, cada demanda é um desafio no qual buscarei concluir da melhor forma.
