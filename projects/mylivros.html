MyLivros App

  • Código Fonte

MyLivros é uma aplicação web projetada para transformar a maneira como você gerencia sua biblioteca pessoal. Desenvolvida como parte de minha jornada de aprendizado em tecnologias de ponta, esta foi minha estreia no uso do Asp.Net Core Mvc, em fevereiro de 2021. Com uma interface intuitiva e recursos eficientes, MyLivros não é apenas um projeto, mas uma demonstração de minha adaptabilidade em tecnologias para criar soluções práticas e relevantes no mundo real.

Features

  • Organizar seus livros por título, autor, gênero e muito mais!
  • Interface amigável e intuitiva
